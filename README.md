# Indian-Housing-Prices-Dashboard
This interactive dashboard provides a comprehensive analysis of Indian housing prices across multiple states, property types, and time periods. It is designed to help users understand pricing trends, regional variations, and affordability patterns in the Indian real estate market.

Key Insights Covered:
Overall Market Snapshot: Displays maximum, minimum, and average housing prices along with the total number of properties analyzed.
Property Type Distribution: Visualizes the proportion of Apartments, Independent Houses, and Villas, offering insights into market composition.
State-wise Price Comparison: Bar charts highlight average property prices across different Indian states, enabling easy regional comparison.
Geographical Analysis: A state-level heat map showcases spatial variations in housing prices across India.

Price Trends Over Time: Line charts track how property prices have evolved based on the year of construction.
Affordability Index: Scatter plots relate average property prices to average property sizes, helping assess affordability across states.

Purpose:
The dashboard is aimed at data-driven decision-making for students, analysts, and real estate enthusiasts by transforming raw housing data into meaningful visual insights.
